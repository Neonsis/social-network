# Social Network

## Инструкция по запуску проекта в docker контейнерах

Для сборки и запуска данного проекта в docker контейнерах на компьютер необходимо установить ТОЛЬКО **docker** и
**docker-compose**.

## Запуск проекта:

###### ! Для Linux систем: Если Вы залогинены в Вашу систему не администратором, то необходимо:

###### 1. Установить переменные окружения:

|   Environment Name   |         
|----------------------|
| CLOUDINARY_NAME      |  
| CLOUDINARY_KEY       |   
| CLOUDINARY_SECRET    |

###### 2. Запустить **build_and_run.sh**

~~~~
sudo sh build_and_run.sh
~~~~

###### 2. Открыть браузер и зайти на сайт:

- Web сайт приложения http://localhost

## Остановка проекта:

###### 1. Запусить **stop.sh**

~~~~
sudo sh stop.sh
~~~~
